# CLAUDE.md - Configuration Directory

Application configuration files managed by chezmoi. Uses chezmoi naming conventions (`private_`, `dot_`, `exact_`, `.tmpl`); see the Chezmoi Expert skill for details.

## Directory Structure

| Directory | Purpose |
|-----------|---------|
| `nvim/` | Neovim editor (see `nvim/CLAUDE.md`) |
| `private_fish/` | Fish shell (experimental) |
| `git/` | Git config, ignore, attributes |
| `mise/` | Tool version manager (`config.toml.tmpl`) |
| `chezmoi/` | Chezmoi self-config |
| `gh-dash/` | GitHub dashboard TUI |
| `starship.toml` | Starship prompt theme |
| `private_kitty/` | Kitty terminal emulator |
| `zellij/` | Terminal multiplexer |
| `private_atuin/` | Shell history manager |
| `ranger/` | File manager |
| `rofi/` | Application launcher (Linux) |
| `sketchybar/` | macOS status bar |
| `private_dunst/` | Notification daemon (Linux) |
| `private_i3/` | i3 window manager (Linux) |
| `fd/` | fd ignore patterns |
| `pet/` | Snippet manager |
